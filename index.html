À propos

Graphiste-intégratrice web depuis 2008, je travaille en collaboration avec les rédactions Web de FRANCE 24, RFI et Monte Carlo Douliya sur toutes formes d'infographies.

Actuellement, j'utilise principalement le HTML 5, CSS 3 et le Jquery pour répondre aux nouveaux usages. Les infographies actuelles produites pour les rédactions Web varient du reportage interactif avec une couche responsive full screen à l’infographie statique. Auparavant, je me servais surtout de flash avec l'AS3 pour les cartes interactives et les animations.

Graphisme Web

Conception et réalisation multimédia online. Maîtrise des logiciels Photoshop, Illustrator, Dreamweaver, Flash. Gestion des contraintes spécifiques au graphisme Web.

Développement Web

Développement et optimisation de templates. Maîtrise des langages : HTML 5, CSS 3, as3. Bonnes connaissances en Jquery.
